## 🌟 Overview

**PolyDoc Chat** is an AI-powered document assistant that enables users to upload multiple documents and interact with them conversationally.

It leverages a modern **RAG (Retrieval-Augmented Generation)** architecture to retrieve relevant content and generate accurate, context-aware answers using LLMs.

---

## 🧠 System Architecture

```

User → Streamlit UI → RAG Pipeline → Vector Database → LLM → Answer

````

### Processing Flow

1. Upload documents  
2. Text chunking  
3. Embedding generation  
4. FAISS vector storage  
5. Semantic retrieval  
6. LLM answer generation
